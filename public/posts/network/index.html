    计算机网络基础 #

    三次握手 #

    1. client 发送 syn（Synchronize Sequence Number）到 server，表示想要建立 tcp 连接
    2. Server 发送(syn+ ack)
    3. client 发送 ack 给 server，连接建立

    四次挥手 #

    1. client 发送 FIN
    2. server 接收到 FIN，发送 ack
    3. server 发送 FIN
    4. client 发送 ack
